# 用户使用指南

## 1. 注册与登录

1. 打开首页。
2. 在登录页输入账号和密码。
3. 如果没有账号，直接使用注册并登录。
4. 账号支持用户名、邮箱或手机号。

## 2. 完善个人资料

1. 进入“个人中心”。
2. 填写身高、体重、目标体重、饮食偏好和健康约束。
3. 保存后返回首页查看推荐变化。
4. 首页会同时显示 BMI、热量目标和蛋白目标。

## 3. 浏览菜谱

1. 进入“菜谱”页面。
2. 浏览菜谱卡片。
3. 后续可在详情页查看营养和收藏状态。

## 4. 记录饮食

1. 进入“记录”页面。
2. 使用“快速录入”提交日期、餐次和菜谱或备注。
3. 查看最近 7 天或 30 天的营养统计。
4. 页面底部会展示趋势明细。

## 5. 查看报表

1. 进入“报表”页面。
2. 生成周报或月报。
3. 点击下载最新 PDF。
4. 报表内容以中文输出，适合保存和提交。

## 6. 社区互动

1. 进入“社区”页面。
2. 发布帖子。
3. 在帖子下评论和交流。

## 7. 推荐理解

1. 首页推荐卡片可以点击。
2. 系统会弹出推荐理由，便于理解推荐逻辑。
