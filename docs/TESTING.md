# 测试清单

## 用户视角测试

- 注册新账号
- 使用账号、邮箱、手机号登录
- 修改个人资料和健康档案
- 新增菜谱并查看营养汇总
- 收藏菜谱并查看推荐变化
- 录入饮食记录并查看周统计
- 使用快速录入保存一条饮食记录
- 创建健康目标并添加进度
- 发布帖子、评论和举报
- 生成周报、月报并下载 PDF
- 查询外部营养库代理接口
- 点击首页推荐卡片查看推荐理由

## 技术测试

- Django 语法检查
- API 认证和权限测试
- 空数据容错测试
- 外部 API 降级测试
- Docker Compose 启动测试
- 执行 `make test` 运行后端烟雾测试，命令会在 Docker 容器内执行
